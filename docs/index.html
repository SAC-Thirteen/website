SAC13 is a thirteen-month solar calendar in which every month has the same four-week layout (28 days ¹). It also fixes many other issues the Gregorian Calendar has, unlike other calendar reform proposals which typically just fix very few issues but still leave a lot of problems behind that could have been easily fixed as well.

If you are here for the first time, the Introduction page is a good starting point. You can also try the calculator if you want to see the calendar in action.
